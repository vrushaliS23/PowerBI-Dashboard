# 🍫 Chocolate Sales Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard provides a comprehensive analysis of chocolate sales performance across products, customers, locations, and time periods.

The dashboard helps businesses track KPIs, identify top-performing products, understand customer behavior, and monitor sales trends.

---

## 🎯 Objectives
- Analyze overall sales performance
- Track revenue and profit KPIs
- Identify top customers and products
- Monitor location-wise sales trends
- Analyze sales over time

---

## 📊 Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %
- Average Order Value

---

## 📈 Dashboard Pages

### Home Page
Provides overall business performance summary and KPI overview.

### Product-wise Analysis
Analyzes product sales, revenue contribution, and profitability.

### Customer-wise Analysis
Identifies top customers and purchasing patterns.

### Location-wise Analysis
Shows regional sales performance and customer distribution.

### Date-wise Analysis
Tracks monthly and yearly sales trends.
